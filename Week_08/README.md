# 一、位运算
## 1、位运算符
#### 1）为什么需要位运算
- 机器里的数字表示方式和存储^格式就是二进制
- 余数短除法除以二
#### 2）位运算符
- 左移  运算符是  <<   示例  0011=>0110
- 右移  运算符   >>  示例 0110=> 0011   或 |， 与 & ，取反 ~
- XOR-异或 
   - 异或：相同为0，不同为1.也可以用“不进位加法”来理解
   - x^0 = x
   - x^1s= ~x //注意 1s = ~0
   - x^(~x) = 1s
   - x^x = 0
   - c = a^b => a^c=b,b^c = a //交换两个数
   - a^b^c = a^(b^b) = (a^b)^c //associative
